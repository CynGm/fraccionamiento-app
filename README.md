# 🏘️ Fraccionamiento App

Sistema integral para la gestión de fraccionamientos, diseñado para colonias privadas, conjuntos residenciales y condominios. Desarrollado con **Python + FastAPI** con enfoque modular y profesional.

---

## 🧩 Módulos Desarrollados

### ✅ Módulo 1: Control de Acceso
- Registro de residentes con QR generado automáticamente
- Descarga de QR por navegador
- Registro de visitas (entrada, salida, placas, pre-registro)
- Historial de visitas vía endpoint
- **(Pendiente):** Reconocimiento facial, lectura de placas y botón de pánico

### ✅ Módulo 2: Gestión de Residentes
- Crear residente con autos, personas y adeudos
- Editar, consultar y eliminar residentes
- Filtros por nombre y correo electrónico
